울릉도 관광 홍보 페이지 기본 템플릿

구성
- index.html: 브라우저에서 바로 열 수 있는 단일 HTML 파일
- README.txt: 사용 방법 안내

사용 방법
1. 이 폴더의 index.html을 더블클릭해 브라우저에서 엽니다.
2. [대괄호]로 표시된 이름, 위치, 대표 메뉴, 문의 방법을 내 정보로 바꿉니다.
3. "내 사진으로 교체" 영역은 직접 찍었거나 사용 권리가 있는 사진으로 바꿉니다.
4. 운영시간, 가격, 배편, 전화번호, 지도 링크는 공개 전에 직접 확인합니다.
5. GitHub Pages에 올릴 때는 index.html 파일 이름을 그대로 유지합니다.

AI에게 수정 요청할 때 쓸 문장
이 index.html을 기준으로 내 관광 홍보 페이지를 완성해줘.
채팅창에 코드를 출력하지 말고, 수정된 index.html과 README.txt가 들어 있는 ZIP 패키지로 만들어줘.
모바일에서 글자와 버튼이 겹치지 않게 하고, 확인이 필요한 정보는 "확인 필요"로 남겨줘.
