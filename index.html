1 개요 · AI 시대

AI는 이제 ‘대답하는 도구’를 넘어
‘일을 처리하는 도구’로

먼저 “요즘 AI가 어디까지 왔는지” 큰 그림을 함께 봅니다. 겁먹을 필요는 없습니다. 중요한 건 이 변화를 울릉도 관광 홍보에 그대로 활용할 수 있다는 점입니다.

01

정부도 공개 전에 AI를 들여다보는 시대

AI가 글쓰기를 넘어 소프트웨어 개발·보안·복잡한 추론까지 하게 되면서, 미국·영국의 AI 안전 기관은 주요 모델을 공개 전에 평가하기 시작했습니다.

“AI가 너무 똑똑해져서, 이제는 정부도 이 AI가 어디까지 할 수 있는지 먼저 확인하는 시대가 되었습니다.”

우리에게는? 무서워하기보다, 안전하고 똑똑하게 활용하는 법을 배우면 됩니다. 그래서 오늘은 사람이 확인·수정하는 법까지 함께 배웁니다.

02

AI가 컴퓨터를 직접 조작하는 시대

예전 AI는 질문에 답만 했습니다. 이제는 화면을 보고, 마우스를 움직이고, 버튼을 누르고, 자료를 찾아 정리하는 사람처럼 컴퓨터를 쓰는 단계로 갑니다.

“예전 AI는 말만 했지만, 이제는 직접 손발이 되어 일을 처리해 줍니다.”

이것이 바로 ‘에이전트 AI’입니다. 접속·검색·입력·정리·수정 같은 반복 작업을 대신 처리합니다.

03

AI가 AI를 조종하는 시대

이제 하나의 AI가 모든 걸 하지 않습니다. 글 작성 AI·리서치 AI·이미지 AI·번역 AI·웹페이지 제작 AI가 역할을 나눠 함께 일합니다.

“예전엔 홍보글·번역·디자인·홈페이지를 따로 했지만, 이제는 순서대로 시키면 하나의 홍보 세트로 연결됩니다.”

오늘 흐름도 이것. Gemini로 글 → 리서치로 조사 → 이미지 → Codex로 웹페이지.

04

검색이 대화형으로 바뀌는 시대

관광객은 이제 “울릉도 맛집” 같은 단어 대신, 질문형·대화형으로 검색합니다.

“부모님과 갈 울릉도 2박3일 코스 알려줘” · “비 오는 날 울릉도에서 갈 만한 곳 추천해줘” · “외국인 친구에게 추천할 울릉도 음식 알려줘”

그래서 홍보도 ‘예쁜 문구’보다 관광객 질문에 잘 답하는 콘텐츠가 중요해졌습니다.

05

문서를 팟캐스트처럼 읽어주는 시대

긴 자료를 넣으면 AI가 두 사람이 대화하듯 핵심을 풀어 설명해 줍니다. 관광 해설·발표 연습에 아주 유용합니다.

“생태관광 해설자료를 넣으면, 오디오 요약으로 만들어 귀로 들으며 연습할 수 있습니다.”

오늘 실습 3에서 NotebookLM으로 직접 만들어 들어봅니다.

06

AI가 외국어 장벽을 낮추는 시대

영어·일본어·중국어 안내문을 훨씬 쉽게 만들 수 있습니다. 특히 “따개비밥·오징어내장탕”처럼 번역만으로는 부족한 음식도, 재료·맛·특징을 함께 풀어 설명해 줍니다.

“외국인이 이해할 수 있게 따개비밥을 설명해줘 — 라고만 해도 됩니다.”

오늘 실습 2에서 다국어 안내문을 직접 만듭니다.

기억할 점: AI는 매우 유용하지만, 결과를 그대로 믿으면 안 됩니다. 가격·영업시간·배편·전화·주소·안전 안내 같은 정보는 반드시 사람이 확인합니다. AI는 초안을 빠르게 만드는 도구이고, 최종 확인과 책임은 사람에게 있습니다.

2 개요 · 우리가 할 수 있는 일 (1/2)

그래서 우리는 이런 것들을 만들 수 있습니다

AI를 활용하면 울릉군민도 직접 관광 홍보 콘텐츠를 만들 수 있습니다. 먼저 글·콘텐츠 계열부터 봅니다. 각 항목은 오늘 또는 돌아가서 바로 써먹을 수 있는 것들입니다.

🖋️관광 홍보 문구

  • 울릉도 한 줄 소개, 계절별·대상별 카피
  • 성인봉·행남해안산책로·봉래폭포 소개
  • 도동항·저동항·사동항 주변 홍보
  • 포스터·현수막·카드뉴스용 짧은 카피
활용 예시 “울릉도를 처음 방문하는 관광객에게 매력을 전달하는 한 줄 소개 10개를 감성적이고 짧게 만들어줘.”

📱SNS 홍보 콘텐츠

  • 인스타그램 글, 블로그 후기형 글
  • 카드뉴스 문구, 릴스·쇼츠 대본
  • 해시태그 묶음, 카카오톡 홍보 메시지
  • 축제·이벤트·체험 모집글
활용 예시 “울릉도 오징어내장탕을 소개하는 인스타그램 홍보글과 해시태그 10개를 만들어줘.”

🖼️홍보 이미지·포스터

  • 관광 포스터, SNS 썸네일, 블로그 대표 이미지
  • 체험 프로그램·행사 홍보 이미지
  • 카드뉴스 이미지 구성안
  • 장면·분위기·색감·비율·문구를 구체적으로 지정
활용 예시 “울릉도 해안 절벽과 푸른 바다가 보이는 세로형 관광 포스터 이미지를 만들어줘. 큰 글씨로 문구 한 줄 포함.”

🔎블로그·검색 노출 콘텐츠

  • 울릉도 여행 준비물, 2박3일 코스 글
  • 성인봉 등산 후기형 글, 맛집·숙소 추천
  • 제목·목차·본문 초안·해시태그 한 번에
  • 관광객이 실제로 검색하는 키워드 반영
활용 예시 “울릉도 맛집을 찾는 관광객이 검색할 만한 블로그 제목 10개를 추천해줘.”

💬FAQ · 고객 응대 문구

  • 배편·날씨·예약·결제·주차 FAQ
  • 예약·취소·환불 안내 메시지
  • 후기 요청·재방문 유도·감사 인사
  • 카톡·문자·DM·네이버 톡톡에 바로 활용
활용 예시 “날씨로 일정이 변경될 때 고객에게 안내하는 정중한 카카오톡 메시지를 만들어줘.”

⭐후기·리뷰 활용

  • 고객 후기 요약, 홍보 포인트 추출
  • 긍정 리뷰 답글·부정 리뷰 정중한 대응
  • 후기 기반 홍보 문구·카드뉴스
  • 부족한 점 분석으로 개선 방향 찾기
활용 예시 “고객 후기 10개를 분석해서 우리 가게의 장점 5가지를 정리해줘.”

🌐다국어 관광 안내문 오늘

활용 예시 “울릉도 해산물 식당 메뉴판을 한·영·일로 만들고, 알레르기 안내 문구도 넣어줘.”

🏡숙소·민박 홍보문

  • 숙소 소개·객실 설명·주변 관광지 안내
  • 예약·체크인·환불 규정 안내문
  • 외국인 고객용 소개문
  • 네이버 예약·블로그·SNS용 홍보글
활용 예시 “울릉도 바다가 보이는 민박집을, 가족 여행객을 대상으로 따뜻하게 소개하는 글을 만들어줘.”

🥾생태관광 체험 프로그램

  • 성인봉 트레킹·해안 산책·약초 탐방·오징어잡이 소개
  • 마을 해설·독도 연계·가족/학생 프로그램
  • 대상·소요시간·참가비·준비물·난이도·주의사항 정리
  • 새 체험 상품 아이디어 기획까지
활용 예시 “성인봉 생태 트레킹 프로그램 소개문을 소요시간·난이도·준비물·예약 안내를 포함해 만들어줘.”

🗺️관광 코스 추천

  • 당일·1박2일·2박3일 코스
  • 가족·등산·맛집·사진 명소 중심 코스
  • 어르신도 부담 없는·비 오는 날 대체 코스
  • 이동·식사·관광·휴식을 묶어 동선으로
활용 예시 “울릉도 2박3일 가족 여행 코스를 이동 동선·식사·관광지·휴식 시간을 포함해 만들어줘.”

📍지도·온라인 정보 정리

  • 영문 상호명·영문 주소·대표 메뉴 영어 설명
  • 지도 등록용 짧은 소개문, 영업·주차·예약 안내
  • 외국인 고객용 설명문, 대표 키워드
  • 인기 검색어·계절 키워드 조사(리서치)
활용 예시 “내 식당의 네이버 지도 영문 소개글과 대표 메뉴 5개 영어 설명을 만들어줘.”

🎓관광 해설·교육 자료

  • 울릉도 자연·관광지 해설문, 해설 스크립트
  • 어린이용 쉬운 설명, 외국인용 영어 해설
  • 안전 안내·오리엔테이션·활동지·퀴즈
  • 발표자료·한 장 요약·발표 대본
활용 예시 “관광 해설사가 성인봉을 소개할 때 쓸 3분 해설문을 쉽게 만들어줘.”

💻웹페이지 제작 오늘

활용 예시 “성인봉 생태 트레킹 체험 예약 페이지를 모바일에서도 보기 좋게 만들어줘.”
에이전트 AI 맛보기

여러 작업을 이어서 처리하기

이번 교육에서는 한 번 질문하고 끝내는 것을 넘어, AI가 여러 단계를 순서대로 이어서 도와주는 방식도 맛봅니다. 예를 들어 이런 흐름입니다.

키워드 조사→경쟁 지역 사례 분석→홍보 포인트 정리→SNS 문구→영·일 번역→이미지 프롬프트→웹페이지 구성
활용 예시 “내 식당 정보를 바탕으로 메뉴판, 영문 소개글, 인스타그램 글, 웹페이지 초안을 순서대로 만들어줘.”
🎯 오늘의 집중 홍보 문구→다국어 안내→웹페이지 완성
이 교육의 핵심: “AI를 한 번 써봤다”가 아니라, 내 일을 AI에게 잘 설명해 결과물을 만들고, 앞으로도 계속 활용하는 힘을 기르는 것입니다.

7 AI 기초 · 프롬프트

프롬프트는 AI에게 주는 ‘부탁 문장’입니다

AI를 잘 쓰는 사람은 컴퓨터를 잘하는 사람이 아니라 또박또박 부탁하는 사람입니다. 좋은 결과는 좋은 부탁에서 나옵니다.

🙁 아쉬운 부탁

“울릉도 홍보글 만들어줘.”
→ 누구를 위해, 무엇을, 어떤 톤인지 알 수 없어 두루뭉술한 답이 나옵니다.

🙂 좋은 부탁

“울릉도를 처음 방문하는 30~50대 가족 여행객에게 보여줄 SNS 홍보글을, 성인봉·해안산책로·오징어 요리를 넣어 따뜻하게 300자 이내로, 해시태그 10개와 함께 만들어줘.”

좋은 부탁에는 이 7가지가 들어갑니다. 하나씩 채워 보세요.

역할을 정해준다

예 “너는 울릉도 관광 홍보 전문 SNS 마케터야.”

대상을 정해준다

예 “울릉도를 처음 방문하는 가족 여행객에게 보여줄 글이야.”

목적을 정해준다

예 “성인봉 트레킹 체험을 신청하고 싶게 만드는 홍보글이야.”

형식을 정해준다

예 “인스타그램 게시글, 300자 이내, 해시태그 10개 포함.”

넣을 내용을 알려준다

예 “성인봉, 원시림, 동해 전망, 왕복 3~4시간, 초보자 가능을 포함해줘.”

하지 말 것도 알려준다

예 “운영시간·가격은 지어내지 말고, 모르면 ‘확인 필요’라고 표시해줘.”

결과는 반드시 확인한다

배편·가격·영업시간·주소·전화·예약 가능 여부·안전 안내·번역·행정 정보는 사람이 직접 확인합니다.
💡 처음부터 완벽하지 않아도 됩니다. 7가지를 다 못 넣어도 괜찮습니다. 결과를 보고 “더 짧게”, “울릉도 느낌 더”, “전화 문구 추가”처럼 다시 부탁하면 점점 좋아집니다.

8 AI 기초 · 페르소나 부여

“너는 ○○○ 전문가야” — 역할을 주면 답이 달라집니다

가장 쉽고 효과가 큰 기술입니다. AI에게 역할(페르소나)을 주면, 그 전문가처럼 말투·지식·관점으로 답합니다.

🙁 역할이 없으면

“울릉도 숙소 소개해줘.”
→ 밋밋하고 일반적인 설명이 나옵니다.

🙂 역할을 주면

“너는 20년 경력의 울릉도 여행 안내자야. 가족 여행객에게 우리 민박을 따뜻하게 소개해줘.”
→ 현지 전문가의 시선으로, 손님을 배려하는 톤이 됩니다.
🎭

상황별 역할 예시

  • “너는 울릉도 관광 홍보 전문가야”
  • “너는 친절한 영어 번역가야”
  • “너는 메뉴판을 잘 만드는 디자이너야”
  • “너는 초등학생도 이해하게 설명하는 해설사야”
➕

역할 + 대상 + 목적

역할에 누구에게(대상)와 무엇을 위해(목적)를 더하면 훨씬 좋아집니다.

“너는 울릉도 생태관광 해설사야(역할). 외국인 가족에게(대상) 안전하게 즐기도록(목적) 성인봉 트레킹을 안내해줘.”

9 AI 기초 · 대화로 다듬기

한 번에 끝내지 않습니다 — 대화하며 다듬기

AI는 대화를 기억합니다. 첫 답이 마음에 안 들어도 괜찮습니다. 이어서 고쳐 달라고 하면 점점 좋아집니다.

1먼저 크게 부탁합니다. “울릉도 민박 소개글 만들어줘.”
2결과를 보고 다듬습니다. “더 짧게.”, “따뜻한 말투로.”, “바다 전망을 더 강조해줘.”
3부분만 고칩니다. “마지막 문단만 다시 써줘. 나머지는 그대로 둬.”
4마음에 들면 사용합니다. 사실 정보는 사람이 확인합니다.
개념 정리

생성형 AI와 에이전트 AI, 무엇이 다를까요?

생성형 AI

물어보면 만들어 줍니다

내가 하나 물으면, 하나를 만들어 줍니다.

질문→답변
예 “울릉도 홍보 문구 5개 만들어줘.”
에이전트 AI

여러 단계를 이어서 처리합니다

목표만 주면, 여러 일을 순서대로 스스로 해냅니다.

조사→정리→작성→번역→웹페이지
예 “키워드 조사하고, 문구 만들고, 웹페이지까지 순서대로 만들어줘.”

10 AI 기초 · 에이전트 사용

에이전트 AI, 이렇게 시킵니다

비결은 간단합니다. 큰 목표를 주고, 할 일을 번호로 나눠 순서대로 시키면 됩니다. 오늘 웹페이지를 만드는 Codex도 이 방식으로 도와줍니다.

아래 다섯 단계를 한 대화에서 이어서 시켜보세요. 앞 단계 결과를 AI가 기억하고 다음에 씁니다.

1

조사

관광객이 무엇으로 검색하는지 먼저 알아봅니다.
울릉도 관광 홍보에 필요한 검색 키워드를 조사한다고 가정하고, 관광객이 검색할 만한 키워드 20개를 추천해줘. 국내 관광객과 외국인 관광객을 나누어 정리해줘.
2

정리

조사한 내용에서 강조할 포인트를 추립니다.
위 키워드를 바탕으로 울릉도 관광 홍보에서 강조해야 할 포인트 5가지를 정리해줘.
3

콘텐츠 만들기

정리한 포인트로 실제 홍보글을 만듭니다.
위 홍보 포인트를 바탕으로 인스타그램 게시글 3개와 블로그 제목 5개를 만들어줘.
4

번역

외국인 관광객을 위해 번역합니다.
위 인스타그램 게시글 중 1개를 영어와 일본어로 자연스럽게 번역해줘.
5

웹페이지 구성

지금까지 만든 것을 페이지 구조로 연결합니다.
지금까지 만든 내용을 바탕으로 울릉도 생태관광 홍보 웹페이지 구조를 만들어줘. 첫 화면, 프로그램 소개, 추천 코스, FAQ, 예약 문의 섹션을 포함해줘.
💡 이것이 에이전트 AI의 기본 흐름입니다. 한 번 답을 받고 끝이 아니라, 조사 → 정리 → 콘텐츠 → 번역 → 웹페이지로 이어집니다. 오늘도 Gemini로 재료를 만들고(생성형), Codex가 웹페이지로 이어갑니다(에이전트). 여러분은 목표를 정하고 확인하는 사람입니다.

11 실전 AI 도구 · 무료로 시작

그래서, 지금 뭘 켜면 되나요?

AI가 좋다는 건 알겠는데, 막상 어떤 걸 열어야 할지 막막합니다. 도구마다 잘하는 일이 조금씩 다릅니다. 아래 도구들은 모두 무료로 시작할 수 있습니다.

💬ChatGPT 가장 범용적인 AI 비서

글쓰기·요약·번역·자료 분석·아이디어 정리까지 두루. 처음 AI를 접할 때 가장 쉽게 시작하기 좋습니다.

  • 관광 홍보 문구·블로그·SNS 글 초안
  • 영어·일본어 안내문, 메뉴 설명
  • 관광객 FAQ, 파일·이미지 읽고 정리
울릉도 예시 “가족 여행객용 SNS 홍보글을 성인봉·해안산책로·오징어 요리를 넣어 따뜻하게 300자로 만들어줘.”
👶 초보자 추천 매우 높음 · ⚠️ 배편·가격·시간은 확인 필요, 무료 사용량 제한 있음

✨Gemini 구글 서비스와 잘 연결

검색 기반 조사·번역·이미지 아이디어에 강점. 관광 키워드와 검색 노출을 고려한 문구에 좋습니다.

  • 울릉도 관광 키워드·경쟁 지역 조사
  • 영어·일본어 번역, 지도용 영문 소개
  • 이미지 생성용 프롬프트 만들기
울릉도 예시 “울릉도 여행객이 검색할 키워드 20개를, 국내·외국인으로 나눠 추천하고 활용법도 알려줘.”
👍 초보자 추천 높음 · ⚠️ 추천 키워드는 네이버·구글 검색과 비교 확인 · 오늘 실습의 기본 도구

📝Claude 긴 글·자료 정리에 강함

긴 문서·발표자료·웹페이지 원고를 체계적으로 구조화하는 데 강합니다. 결과를 보기 좋게 확인하는 기능도 있습니다.

  • 체험 프로그램 소개문·관광 해설 스크립트
  • 발표자료 원고·수업자료 목차
  • 웹페이지 문안 정리, HTML 초안
울릉도 예시 “울릉도 생태관광 체험 프로그램 소개 페이지 원고를, 제목·소개·카드·FAQ·예약 안내로 만들어줘.”
👍 초보자 추천 높음 · ⚠️ 무료 사용량 제한, 최종 정보는 사람이 확인

🧑‍💻Codex 웹페이지·코드 제작 도움

코딩을 잘 몰라도 웹페이지 제작·수정을 도와주는 AI 코딩 도구. “글 → 웹페이지” 단계를 보여주기 좋습니다.

  • HTML 웹페이지 초안 만들기·수정
  • 버튼 색·FAQ 아코디언·모바일 반응형
  • 오류 메시지 해석, 코드 쉽게 설명받기
울릉도 예시 “울릉도 체험 프로그램 홍보 HTML 페이지를 첫화면·소개·준비물·FAQ·예약 폼과 함께 만들어줘.”
🙂 초보자 추천 중간 · ⚠️ 완전 초보에겐 화면이 어려울 수 있음 — “맛보기” 수준으로

🤖GitHub Copilot 개발자용 코드 보조

웹페이지·앱을 만들 때 코드 작성·수정·디버깅을 돕습니다. 더 해보고 싶은 분을 위한 다음 단계입니다.

  • HTML/CSS 코드 추천·UI 코드 만들기

  • 오류 설명, 코드 의미 설명받기
